# Number Guessing Game Project

This project is a Bash script that runs a Number Guessing Game connected to a PostgreSQL database. It was built to satisfy the FreeCodeCamp Relational Database Certification requirements. The script generates a random number between 1 and 1000 for users to guess, stores usernames and game results in a database, and follows the exact user stories defined by FreeCodeCamp.

To set up, create a database named `number_guess` by connecting to the interactive shell with `psql --username=freecodecamp --dbname=postgres` and running `CREATE DATABASE number_guess;`. Inside your script, define the PSQL variable as `PSQL="psql --username=freecodecamp --dbname=number_guess -t --no-align -c"`. Place the script in a folder named `number_guessing_game` inside the project directory, name it `number_guess.sh`, and give it executable permissions with `chmod +x number_guess.sh`. Turn the folder into a Git repository with at least five commits. The first commit message must be `Initial commit`, and the rest must start with `fix:`, `feat:`, `refactor:`, `chore:`, or `test:`. The final state must be on the `main` branch with a clean working tree and no uncommitted changes.

When the script runs, it should prompt the user with `Enter your username:` and take a username as input. The database must allow usernames up to 22 characters. If the username has been used before, the script should print `Welcome back, <username>! You have played <games_played> games, and your best game took <best_game> guesses.` where `<username>` is the user’s name from the database, `<games_played>` is the total number of games that user has played, and `<best_game>` is the fewest number of guesses it took that user to win. If the username has not been used before, the script should print `Welcome, <username>! It looks like this is your first time here.`

After greeting, the script should print `Guess the secret number between 1 and 1000:` and read input from the user. Until the secret number is guessed, the script should print `It's lower than that, guess again:` if the previous input was higher than the secret number, and `It's higher than that, guess again:` if the previous input was lower. If anything other than an integer is input, it should print `That is not an integer, guess again:`. When the secret number is guessed, the script should print `You guessed it in <number_of_guesses> tries. The secret number was <secret_number>. Nice job!` and finish running. The game result (`user_id`, `attempts`) must be saved to the `games` table in the database.

FreeCodeCamp’s tests will add users to your database automatically when the script has that ability, so extra usernames may appear. This is expected. To save progress, you can dump your database with `pg_dump -cC --inserts -U freecodecamp number_guess > number_guess.sql` and rebuild it with `psql -U postgres < number_guess.sql`. Submit both `number_guess.sql` and `number_guess.sh` in a public repository on FreeCodeCamp.org once all tests pass.
